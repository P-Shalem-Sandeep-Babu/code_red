# Bhaswara 2k26 - Code Red

**19th National Level Technical and Management Fest - Stranger Things Edition**

Welcome to the official repository for the **Bhaswara 2k26 - Code Red** event website. This immersive web application is designed with a "Stranger Things" theme, featuring an interactive "Upside Down" dimension toggle, audio effects, and smooth animations.

## 🌟 Features

- **Dual Dimension Interface:** Toggle between the "Real World" and the "Upside Down" with a custom switch.
- **Immersive Audio:** Background ambient sounds and interactive sound effects (click, beep, static).
- **Interactive Animations:** Powered by `framer-motion` for smooth transitions and scroll effects.
- **Event Information:**
  - **Challenge Levels:** Details about the technical challenges (Compass Glitch, Blindfold Retrieval, The Light Search).
  - **Leaderboard:** Arcade-style high score display for prizes.
  - **Countdown Timer:** Ticking down to the event launch.
  - **Registration Portal:** Direct link to the registration form.
  - **Coordinators:** Contact information for faculty and student coordinators.

## 🛠️ Tech Stack

- **Framework:** React
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React

## 🚀 Run Locally

**Prerequisites:** Node.js installed on your machine.

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Set up Environment Variables:**
    If you plan to use Gemini API features, set the `GEMINI_API_KEY` in .env.local.

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open in Browser:**
    Navigate to the local URL provided in the terminal (usually `http://localhost:5173`).
